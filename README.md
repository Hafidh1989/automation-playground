
# automation-playground

A unified Python-based test automation portfolio covering **API**, **Web**, and **Mobile** testing.

This repository showcases end-to-end automation skills across multiple platforms using modern tools and scalable test architecture.

## Tech Stack

- **Language:** Python
- **API Testing:** Pytest
- **Web Automation:** Playwright
- **Mobile Automation:** Appium
- **Test Runner:** Pytest
- **Reporting:** Allure 
- **CI/CD:** GitHub Actions

## Project Scope

This repository is designed to demonstrate automation coverage for:

- **API testing**
  - Endpoint validation
  - Response schema validation
  - Authentication flows
  - Negative test cases
  - Data-driven testing

- **Web testing**
  - UI validation
  - End-to-end user flows
  - Form validation
  - Cross-page navigation
  - Assertions for functional behavior

- **Mobile testing**
  - Android app automation
  - UI interaction testing
  - Functional flow validation
  - Deep link testing
  - Native app behavior checks
